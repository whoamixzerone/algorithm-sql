### 출처 LeetCode

#### 시간 복잡도
**n(1 <= num <= 10^6)**  
$`O(logn)`$

#### 문제 유형
시뮬레이션

#### 문제 풀이

### c++ 풀이
```c++
class Solution {
public:
    int numberOfSteps(int num) {
        int answer=0;
        while(num) {
            if(num%2==0) num>>=1;
            else num--;
            answer++;
        }
        return answer;
    }
};
```
