#### 문제 풀이

### c++ 풀이
```c++
class Solution {
public:
    int maximumWealth(vector<vector<int>>& accounts) {
        int mx = 0;
        for(auto account : accounts) {
            mx = max(mx, accumulate(account.begin(), account.end(), 0));            
        }
        return mx;
    }
};
```
