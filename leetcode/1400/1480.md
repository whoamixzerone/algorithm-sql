#### 시간 복잡도
**N(1 ≤ nums.length ≤ 1,000)**  
$`O(N)`$

#### 문제 유형
구현  
구간합(prefixsum)

#### 문제 풀이

### c++ 풀이
```c++
class Solution {
public:
    vector<int> runningSum(vector<int>& nums) {        
        for(int i=1; i<nums.size(); ++i){
            nums[i]+=nums[i-1];
        }
        return nums;
    }
};
```
